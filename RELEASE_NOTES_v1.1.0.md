# Ada Chat v1.1.0 发布说明 / Release Notes

发布日期：2026-03-09

## 重点更新
- 统一了跨任务“专业引导话术”：聊天路由到文生图/编程/OCR/视觉/翻译时，先说明调用模型，再返回结果。
- 生成图片结果支持点击大图预览，新增保存/分享操作，并支持会话内多图切换（上一张/下一张）。
- 结果元信息（模型/供应商）改为跟随 AI 结果气泡显示，不再显示在用户提示词气泡。
- 预设绑定模型交互升级：保存后保持当前预设视图，已绑定但暂不可用模型会保留显示。
- 对话列表策略升级：重新激活历史会话或有新消息时自动置顶。
- 结构化回答渲染升级：重点内容块级高亮与复制，中文/英文重点段落均可命中。

## 主要修复
- 修复文本误判为图片 URL 导致浏览器发起错误 GET 并出现 404。
- 修复预设模型下拉“保存后回到自动”的视觉误导问题。
- 修复中文重点段落在双语提示词场景下提取失败问题。

## 升级建议
- 升级后建议强制刷新页面（`Ctrl+F5`）以清除旧 JS 缓存。
- 若新增 Node 后终端仍无法识别命令，请重开终端会话。

## 发行包
- Windows：`AdaChat_Setup_v1.1.0.exe`
- Linux/macOS：`AdaChat_v1.1.0.tar.gz`
