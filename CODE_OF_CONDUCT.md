# Code of Conduct

## Our Commitment

为了构建友好、专业的协作环境，所有参与者应尊重彼此，聚焦技术问题与建设性沟通。

## Expected Behavior

- 尊重不同背景与观点；
- 对事不对人，避免攻击性言论；
- 接受并提供建设性反馈；
- 报告问题时提供清晰上下文与复现信息。

## Unacceptable Behavior

- 人身攻击、骚扰、歧视性言论；
- 恶意刷屏、挑衅、引战；
- 泄露他人隐私或敏感信息；
- 明知违规仍反复行为。

## Enforcement

维护者有权对违规行为采取提醒、限制互动、关闭讨论或拒绝贡献等措施。

## Scope

本准则适用于仓库 issue、PR、讨论区以及与项目相关的协作渠道。
