# Ada Chat v1.1.1 发布说明 / Release Notes

发布日期：2026-03-18

## 重点更新
- 新增视频异步任务轮询：视频接口返回 `task_id` 时，前端自动轮询任务状态并在完成后自动回填视频地址。
- 新增“仅保留图像模型”一键筛选：在供应商模型管理中可快速取消非图像模型勾选，提升图像场景操作效率。
- 完成新增功能 i18n 对齐：高级参数 JSON、视频任务状态提示、模型获取与手动添加提示均提供中英双语文案。

## 工程与发布
- 版本号统一升级为 `v1.1.1`（前端版本检查、安装器版本、文档入口同步更新）。
- 主仓库明确排除 `.updater/` 与 `cyoa/`，避免本地镜像仓库和独立模块误入版本控制。

## 发行包
- Windows：`AdaChat_Setup_v1.1.1.exe`
- Linux/macOS：`AdaChat_v1.1.1.tar.gz`
