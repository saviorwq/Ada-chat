# 发布说明 v1.1.0 / Release Notes v1.1.0

发布日期：2026-03-09

## 本版重点 / Highlights
- 跨任务专业引导话术：聊天自动路由到文生图/编程/OCR/视觉/翻译时，先说明“调用了哪个模型处理何种任务”，再给出结果。
- 生成图预览能力：支持点击图片进入大图预览，左上角关闭、右上角保存/分享。
- 会话内图片相册：预览层支持上一张/下一张按钮与键盘左右键切换。
- 元信息归位：模型与供应商信息跟随 AI 结果气泡展示，不再挂在用户输入气泡。
- 结构化重点提取：中英文提示词分块渲染，支持块级复制。

## 稳定性修复 / Fixes
- 修复文本误识别为图片 URL 导致 404 的问题。
- 修复预设绑定模型保存后界面看似回退“自动”的问题。
- 修复双语内容场景下中文重点段落命中不稳定的问题。

## 升级提醒 / Upgrade Notes
- 升级后请 `Ctrl+F5` 强制刷新页面，以避免旧缓存脚本影响体验。
- 若终端新装 Node 后命令未生效，请重开终端会话。
