# 发布说明 v1.1.1 / Release Notes v1.1.1

发布日期：2026-03-18

## 本版重点 / Highlights
- 视频异步任务轮询：当视频接口先返回 `task_id` 时，前端会自动轮询状态并在完成后自动显示视频地址。
- 新增“仅保留图像模型”按钮：在模型勾选区一键取消非图像模型，便于图像任务快速整理模型清单。
- 新增功能文案 i18n 对齐：高级参数、视频任务提示、模型获取提示等新增字段全部接入中英双语。

## 工程维护 / Maintenance
- 版本统一提升至 `v1.1.1`。
- 主仓库忽略本地更新镜像 `.updater/` 和独立仓库 `cyoa/`，防止误提交。

## 升级提醒 / Upgrade Notes
- 升级后请 `Ctrl+F5` 强制刷新页面，以避免旧缓存脚本影响体验。
