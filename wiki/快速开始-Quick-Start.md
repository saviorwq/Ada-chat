# 快速开始 / Quick Start

适合“已经懂 API 配置”的用户，5 分钟走完。  
For users familiar with API setup. Finish in ~5 minutes.

## 1) 下载 / Download
- Windows: `AdaChat_Setup_v1.0.3.exe`
- Linux/macOS: `AdaChat_v1.0.3.tar.gz`

## 2) 登录 / Login
- 默认密码：`admin123`

## 3) 配置供应商 / Configure Provider
- 填 Base URL + API Key
- 点击 `获取最新模型`
- 勾选并 `保存模型选择`

## 4) 开始对话 / Start Chat
- 选择分类/供应商/模型
- 发送测试消息
