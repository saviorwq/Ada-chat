# 功能说明 / Feature Guide

## 对话区主流程
1. 先选分类（对话/编程/图像等）
2. 再选供应商与模型
3. 输入问题并发送
4. 根据回复继续追问（系统会自动带上历史）

## 任务分类
- `💬 对话`：通用问答
- `💻 编程`：代码生成/修复
- `🎨 图像生成`：文生图/图生图
- `📄 OCR`：提取图片/PDF文字
- `👁️ Vision`：看图分析
- `🌐 翻译`：文本或图片翻译

## 设置页核心
- 供应商与模型：新增供应商、拉取模型、保存选择
- 对话能力：预设、RAG、模式矩阵、单词转换、聊天身份、模型总设置
- 界面系统：语言、皮肤、成本优化、自动切换
- 安全开发：密码、插件、调试

## 模型总设置（新手重点）
- `temperature`：控制“发散度”。越低越稳，越高越有创意。
- `top_p`：控制“采样范围”。一般和 `temperature` 二选一微调即可。
- `max_tokens`：单次回复最大长度上限。留空表示走模型默认/系统策略。
- `presence_penalty`：鼓励聊“新点子”。值高时更容易跳出原话题。
- `frequency_penalty`：减少重复表达。值高时更不容易复读。
- `stop`：命中终止词后立刻停止输出（每行一个）。

### 新手推荐起步值
- 通用聊天：`temperature=0.7`，`top_p=1`，其余保持默认
- 写代码：`temperature=0.2~0.5`，减少随机性
- 创意文案：`temperature=0.8~1.0`，可适当提高发散

## 回答下方快捷操作
- `📋` 复制整条回答
- `🔄` 重回答（含文生图场景）
